# is.TYPE

What's type of the Object? Let me tell you.

[![Build Status](https://secure.travis-ci.org/jsw0528/is.png)](http://travis-ci.org/jsw0528/is)

---

## Installation

```sh
npm install is
```

## Usage

```js
var is = require('is')

is.Array([])  // true
is.Object([]) // false
is.Empty([])  // true

// more
console.log(is)
```
